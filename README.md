# CPP-AR-Development-Project

Augmented Reality program for the HoloLens

Users are asked to create simple molecules by interacting with element spheres. 

## Requirements
* Unity 2017.4.30
* C#
* Visual Studio

## Technologies Used
* C# 
* Visual Studio 
* HoloLens AR.
